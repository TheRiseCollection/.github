## What We Do

THE RISE COLLECTION is a software company that builds products across **iOS**, **Android**, **web**, and **gaming platforms**. We create arcade-style games, developer tools, 3D assets, financial APIs, and digital brands—from concept to storefront.

### Our work spans

| Platform | Products |
|----------|----------|
| Mobile & iOS | BL1NK, Stack 8, Keep On Trucking, Attack Of The Circle, Baaridge |
| Developer Tools | devKick, styleSweeper, processLogger |
| Financial APIs | Dividend Chaser, Dividend Chaser API |
| 3D Assets | Unity Asset Store & Unreal Engine marketplace |
| Web & Brands | redlightcam®, Point Five, controller-chat |
